# Homebrew Tap for CRAGuard

EU Cyber Resilience Act compliance scanner for embedded firmware.

## Install

```bash
brew tap shieldmark-io/craguard
brew install craguard
```

## Usage

```bash
craguard scan ./firmware-project
craguard report --format html ./firmware-project
```
